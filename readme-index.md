# cowork-research: Team Sonoma Fyre

Single source of truth for our Applied Innovation Immersion Week research (reversing declining wine demand, Sonoma). Each teammate connects their own Claude to this shared folder through the Google Drive connector, so we all work from the same knowledge. Keep everything we want shared inside this folder.

## Folder map
- `CLAUDE.md`: the team working rules. Every member's Claude reads and follows it; teammates should skim it once.
- `master-synthesis.md`: the one doc where findings converge. Start here to see where we stand.
- `ai-tools-log.md`: our running AI-tools log (a graded requirement).
- `deliverables/`: graded outputs (project outline, final deck, research results).
- `research-readings/`: sources and desk research (market, health and GLP-1, regional economics). Keep a running sources list here.
- `interviews/`: interview guides and per-interviewee notes.
- `consumer-field/`: run-club and tasting notes; anything from talking to actual drinkers.

## Working rules
All conventions live in `CLAUDE.md`: source of truth, naming, citations, the AI-tools log, style, and the merge protocol for `master-synthesis.md`. Ask research questions in your own Claude space; it pulls from this folder and follows those rules. If your Claude seems unaware of them, redo Step 2.8 in `setup-guide.md`.
